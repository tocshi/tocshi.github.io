Bouncing Balls Project

This Adobe Flash project was the second frame-based animation I made this year. I used both frame-by-frame animation along with motion tweens to create a smooth animation. I honestly found the project too easy, so I decided to add some extra context to it, hence the Animator vs. Animation style.


Toolbar Comparison Project

This Adobe Flash project gave me my very first experience with buttons. I felt that including only hover-buttons for the tool symbols was too simple, so I decided to add a toggle feature that switches between two different toolbars.


Colouring Book Project

The Colouring Book assignment was one of my biggest Adobe Flash feats this semester. It involved using buttons, replacing the cursor, and changing the colours of vectors. This colouring book works by clicking a colour on the colour palette and then clicking an area in the picture to replace the colour in that picture with the one that you currently have selected. The two pictures that I used for the colouring book were both drawn by me.
